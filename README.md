# 🛍️ Full-Stack E-Commerce App (Spring Boot + Flutter)

## 🚀 Overview
A multi-vendor e-commerce platform with mobile app (Flutter) and backend (Spring Boot).

## 🧱 Tech Stack
- Backend: Spring Boot, PostgreSQL, JWT
- Mobile: Flutter, Riverpod
- DevOps: Docker, GitHub Actions

## 📦 Features
- Authentication (JWT)
- Product management
- Cart & orders
- Payment integration
